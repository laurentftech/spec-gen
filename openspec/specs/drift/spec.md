### Requirement: GitDiffValidation

The system SHALL validate GitDiff according to these rules:
- maxChars must be a positive integer
- baseRef must be a valid Git reference (commit, branch, tag)
- filePath must be a valid file path within the repository

#### Scenario: GetFileDiff
- **GIVEN** A Git repository with changes between the base reference and the current state
- **WHEN** getFileDiff() is called with valid parameters
- **THEN** The unified diff content for the specified file is returned, truncated to maxChars if necessary

#### Scenario: GetChangedFiles
- **GIVEN** A Git repository with changes between the base reference and the current state
- **WHEN** getChangedFiles() is called with valid parameters
- **THEN** A list of changed files with their statuses is returned
