# Verifier Specification

> Generated by spec-gen v1.0.0 on 2026-03-09 23:34
> Source files: src/core/verifier/verification-engine.ts

## Purpose

Verifies code files against specifications to ensure they meet requirements and maintain
architectural integrity.

## Entities

### VerificationCandidate

Represents a file selected for verification against the specification.

**Properties:**

| Name | Type | Description |
|------|------|-------------|
| filePath | string | Path to the file |
| domain | string | Domain the file belongs to |

**Relationships:**

- has-one VerificationResult (Verification result for this candidate)

### FilePrediction

LLM-generated prediction about a file's purpose and content.

**Properties:**

| Name | Type | Description |
|------|------|-------------|
| purpose | string | Predicted purpose of the file |
| imports | string[] | Predicted imports |
| exports | string[] | Predicted exports |

**Relationships:**

- belongs to VerificationCandidate (Prediction is for this candidate)

### PurposeMatch

Result of comparing a file's actual purpose with the specification.

**Properties:**

| Name | Type | Description |
|------|------|-------------|
| score | number | Similarity score between 0 and 1 |
| keywords | string[] | Common keywords between actual and expected |

**Relationships:**

- belongs to VerificationResult (Match result is part of this verification)

### SetMatch

Result of comparing a file's imports/exports with the specification.

**Properties:**

| Name | Type | Description |
|------|------|-------------|
| score | number | Similarity score between 0 and 1 |
| missing | string[] | Items missing from the file |
| extra | string[] | Extra items in the file |

**Relationships:**

- belongs to VerificationResult (Match result is part of this verification)

### RequirementCoverage

Analysis of how well a file covers the specified requirements.

**Properties:**

| Name | Type | Description |
|------|------|-------------|
| score | number | Coverage score between 0 and 1 |
| covered | string[] | Requirements covered by the file |
| uncovered | string[] | Requirements not covered by the file |

**Relationships:**

- belongs to VerificationResult (Coverage analysis is part of this verification)

### VerificationResult

Result of verifying a single file against the specification.

**Properties:**

| Name | Type | Description |
|------|------|-------------|
| filePath | string | Path to the verified file |
| purposeMatch | PurposeMatch | Purpose match result |
| importsMatch | SetMatch | Imports match result |
| exportsMatch | SetMatch | Exports match result |
| requirementCoverage | RequirementCoverage | Requirement coverage analysis |
| overallScore | number | Overall verification score between 0 and 1 |
| feedback | SuggestedImprovement[] | Suggested improvements |

**Relationships:**

- belongs to VerificationCandidate (Result is for this candidate)
- belongs to VerificationReport (Result is part of this report)

### DomainBreakdown

Breakdown of verification results by domain.

**Properties:**

| Name | Type | Description |
|------|------|-------------|
| domain | string | Domain name |
| averageScore | number | Average verification score for the domain |
| fileCount | number | Number of files in the domain |

**Relationships:**

- belongs to VerificationReport (Breakdown is part of this report)

### SuggestedImprovement

Suggested improvement for a file.

**Properties:**

| Name | Type | Description |
|------|------|-------------|
| type | string | Type of improvement |
| message | string | Improvement message |
| severity | string | Severity level |

**Relationships:**

- belongs to VerificationResult (Improvement is for this verification result)

### VerificationReport

Complete verification report for all files.

**Properties:**

| Name | Type | Description |
|------|------|-------------|
| results | VerificationResult[] | Verification results for all files |
| domainBreakdown | DomainBreakdown[] | Breakdown by domain |
| overallScore | number | Overall verification score between 0 and 1 |

### VerificationEngineOptions

Options for configuring the verification engine.

**Properties:**

| Name | Type | Description |
|------|------|-------------|
| thresholds | object | Thresholds for different verification aspects |
| outputFormat | string | Output format for reports |
